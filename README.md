# Victory Air Solutions Website

This is a static website ready for GitHub Pages.

## Files
- index.html
- style.css
- script.js
- assets/logo.png

## Publish with GitHub Pages
1. Create a GitHub account or sign in.
2. Create a new public repository named `victory-air-solutions`.
3. Upload all files in this folder.
4. Go to Settings > Pages.
5. Under Source, choose `Deploy from a branch`.
6. Choose branch `main` and folder `/root`.
7. Click Save.
8. Your site will publish at `https://YOUR-USERNAME.github.io/victory-air-solutions/`.
